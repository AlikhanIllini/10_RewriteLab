# RewriteLab Wireframes

This folder contains UI/UX wireframes for the RewriteLab application.

## Structure
- `v1/` - Initial wireframe iteration

## Planned Screens
1. **Home / Rewrite Page** - Main text input with context and tone selectors
2. **Results / Compare Page** - Side-by-side comparison of original and rewrites
3. **Help / Tips Page** - User guidance and examples

## Design Principles
- Clean, minimal interface
- Focus on the text content
- Easy comparison between versions
- Mobile-responsive layout

*Wireframes will be added as the project progresses.*
